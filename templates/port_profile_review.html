ICS Port Profile Review

Confirm the profile-defined audio and radio control mapping for each enabled port.

This ICS hardware profile defines the audio, PTT and COS mapping. The normal single-port interface selection page is skipped because these choices are already fixed by the selected board profile.

At this stage SvxLink will use only the PTT and COS mappings. EN and CT lines remain hardware metadata unless later configuration makes use of them.
