Editable settings after build

Some dashboard settings can still be edited later at the discretion of the builder. These include programmable talkgroup buttons, monitored talkgroups, audio level maintenance, transmit and receive calibration, later addition or adjustment of EchoLink and MetarInfo settings, and the node location and display information.

The initial build creates the working SvxLink configuration from the choices made so far, but these editable dashboard sections remain available for later refinement. For further information, please refer to the detailed guidance in the manual.

Caution: The SvxLink configuration files are not intended to be edited directly.
Any changes made outside of the dashboard may be overwritten by future builds.
Consider backing up your configuration made with any manual changes.
