Port Identification

Configure identification behaviour for each enabled ICS port.

Each enabled port is treated as its own logical SvxLink node. The callsign was set earlier; this page controls how and when each port identifies itself.

Short and long ident settings are stored separately because SvxLink allows voice, CW and announcement-file identification to be enabled independently.
