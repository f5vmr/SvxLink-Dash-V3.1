Node Type

Choose the type of SvxLink station you are building.
A simplex node is a standalone station that transmits and receives on the same frequency,
while a repeater node is connected to a radio transmitter and receiver and can be used with a
duplexer to operate on separate transmit and receive frequencies.

The node type can be changed later if needed, but the callsign cannot be changed after this step.
