This system includes an operational dashboard for viewing live SvxLink status and controlling selected runtime functions.

The public dashboard view can show node status, reflector activity, and talkgroup buttons. Configuration and editing functions are protected.

Create a dashboard username and password now. These credentials will later be required to edit EchoLink, METAR, talkgroups, monitoring talkgroups, and other runtime configuration pages.

If the credentials are forgotten, they can be reset locally from the Linux console.
